## Contributing
We welcome contributions, make streaming accessbile for everyone !
Please check the `issues` tab to find open tasks.  
Follow these steps:
1. Fork the repo
2. Create a new branch: `git checkout -b fixed addons`
3. Commit changes: `git commit -m "Xyz Addons added"`
4. Push to your fork and create a Pull Request

# Contributing Guidelines
- Use clear commit messages, which addons added, which addons fixed.
- Create separate branches for each feature.
- Ensure code passes all tests before creating PRs.
